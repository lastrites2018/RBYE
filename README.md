# RBYE

Requirements By Years of Experience

연차별로 요구되는 능력을 빠르게 파악하기 위해서 만들었습니다.

백엔드 데이터는 비정기적으로 업데이트 되기에 최신 데이터가 아닐 수 있습니다.
구직의 목적이라면 원티드 사이트를 직접 방문하시길 권장합니다.
